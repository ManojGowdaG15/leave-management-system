# Leave Management System

A comprehensive Leave and Expense Management System built with MERN stack.

## Features

### For Employees:
- Apply for leave with date selection and leave type
- View leave application history and status
- Check leave balance (Casual, Sick, Earned)
- Cancel pending leave applications
- User authentication and role-based access
- Responsive dashboard

### For Managers:
- View team leave requests (pending approvals)
- Approve or reject leave applications with comments
- View team leave calendar
- View leave history of team members
- Manager dashboard with analytics

## Technology Stack

### Frontend:
- React 18
- React Router DOM
- Axios for API calls
- Tailwind CSS for styling
- React Icons
- Chart.js for data visualization

### Backend:
- Node.js
- Express.js
- MongoDB with Mongoose
- JWT for authentication
- bcryptjs for password hashing

## Prerequisites

- Node.js (v14 or higher)
- MongoDB Atlas account or local MongoDB
- npm or yarn package manager
