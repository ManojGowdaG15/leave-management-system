The frontend is built using **React**, **Vite**, **Axios**, and **Context API**.  
It offers dashboards for employees, managers, HR, and admin, with dynamic data loading from the backend.

---

##  Features
- Employee Dashboard (leave balance, history, apply)
- Manager Dashboard (pending approvals)
- HR Dashboard (system-wide visibility)
- Admin Dashboard (full control)
- Protected routes
- JWT-based authorization
- Axios interceptors for auto token handling

---

##  Environment Variables

Create `.env` in frontend folder:

VITE_API_URL=https://your-backend.onrender.com

---

##  Installation
Install dependencies:
npm install
